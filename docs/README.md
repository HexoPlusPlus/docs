HexoPlusPlus~~也叫Hexo艹~~ 或者叫HPP、Hexo++，是一个基于`CloudFlareWorkers`和`CloudFlareKV`技术的Hexo后端程序，目的是解决Hexo无后端带来的种种麻烦，支持但不限于Hexo等静态博客。



**注意，HexoPlusPlus在使用前您的博客必须保证已完成集成部署**

## 特色

- 免费
- 易上手
- 图形化安装，KV数据库
- 登录验证码
- 多用户登录
- EditorMD编辑器
- 全主题适配
- Material Dashboard，细腻书写新体验

## 功能

- 文章在线编辑
- Github图床支持
- 文件管理
- 活跃时间统计
- 说说功能
- Twikoo加强

点击快速上手，开始使用
