# API

# 非鉴权API

## `/hpp/api/getblogeractive` `GET`

获取博主活跃时间，默认以js输出，可以直接以`<script src="https://{yourdomain}/hpp/api/getblogeractive"></script>`使用，此种方法会自动替换id为bloggeractivetime的html的元素

返回实例：

```js
document.getElementById("bloggeractivetime").innerHTML='博主在39小时前活跃了一次'
```

# 鉴权API


## `/hpp/admin/api/kick` `GET`

博主手动签到，点击右上角小飞机，以任意性形式访问都能够刷新活跃时间。

返回：

```
OK
```

## `/hpp/admin/api/adddoc/` `POST`

增加文档，/后面是文件名，以post提交base64后的文件既可。

返回值与Github返回值相同。

## `/hpp/admin/api/addimage/` `POST`

增加图片，/后面是后缀名，以post提交base64后的文件既可
返回值是jsd加速连接。


## `/hpp/admin/api/deldoc/` & `/hpp/admin/api/delimage/` `POST`

删除图片/文档，/后面是文件名。
返回值：
```
Delete Success
``` 
返回状态与Github返回状态相同

## `/hpp/admin/api/getdoc/` `GET`
获取文档，此处不用jsd获取的原因是缓存更新太慢，所以用CloudFlare中间件中继
